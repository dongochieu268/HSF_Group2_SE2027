Phase 1

Recruitment Web System

Khung ứng dụng Spring MVC ba lớp đã sẵn sàng để phát triển các nghiệp vụ tuyển dụng.

Xin chào Người dùng, vai trò CANDIDATE.

Công ty

Quản lý công ty và hồ sơ doanh nghiệp.

Việc làm

Đăng và theo dõi tin tuyển dụng.

Ứng viên

Quản lý hồ sơ và kỹ năng ứng viên.

Ứng tuyển

Theo dõi application và lịch phỏng vấn.

Cấu hình Phase 1

  • Java 17 và Spring Boot 3.2
  • Spring MVC, Thymeleaf, JPA và Validation
  • SQL Server với Hibernate ddl-auto=update
  • BCrypt PasswordEncoder, không bật Spring Security
  • Không sử dụng Lombok
  • Đăng ký, đăng nhập và phân quyền session ở Phase 2
